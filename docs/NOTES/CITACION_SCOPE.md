Citation Scope

This repository contains multiple forms of research output related to the Akimov Theory of Unity (ToU).

To ensure consistency and accuracy in academic, technical, and analytical references, the following citation guidelines are recommended:

1. Formal Framework

For references to the formal theoretical structure, cite the Zenodo record of the ToU Framework (v1.x series).

This represents the primary scientific articulation of the model.

2. Repository and Implementation

For references to implementation details, simulation structure, or technical organization, cite the GitHub repository and the corresponding release version.

3. Author Statement

For references to interpretative positioning, research intent, or conceptual framing, cite the Author Statement document.

4. General Recommendation

If in doubt, prioritize citation of the formal framework document, supplemented by repository references where necessary.

---

This document provides guidance for citation clarity. It is not a legal requirement but a recommended standard for academic and technical consistency.
